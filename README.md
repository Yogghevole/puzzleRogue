# PuzzleRogue

**PuzzleRogue** è un videogioco single-player che fonde la logica classica del Sudoku con meccaniche di progressione e gestione risorse tipiche del genere Roguelike.

## Prerequisiti

Per eseguire il gioco è necessario avere installato sul proprio computer:
*   **Java Runtime Environment (JRE) 17** o superiore.

## Installazione e Avvio

1.  Scarica il file **`puzzleRogue.jar`** presente in questa cartella.
2.  Per avviare il gioco, puoi:
    *   Fare doppio clic sul file `puzzleRogue.jar`.

## Guida

### Accesso
All'avvio, inserisci il tuo **Nickname**:
*   Se esiste già, verranno caricati i tuoi progressi.
*   Se è nuovo, verrà creato un nuovo profilo.

### Menu Principale
*   **Venture Forth**: Continua l'ultima spedizione salvata.
*   **New Expedition**: Inizia una nuova partita (Run).
*   **The Ancestor's Legacy**: Accedi al negozio per acquistare **Buff Permanenti** (Vite Extra, Protezione Errori, ecc.) usando i punti accumulati.
*   **Impostazioni (Ingranaggio)**: Regola audio, fai Log Out o chiudi il gioco.

### Controlli di Gioco
*   **Selezione**: Clicca su una cella per selezionarla.
*   **Inserimento**: Premi i tasti `1`-`9` o usa la barra numerica in basso.
*   **Gomma**: Usa l'icona della gomma per cancellare numeri o note.
*   **Modalità Note (Matita)**: Attiva la matita per inserire piccoli numeri (appunti) che non contano come errori.

### Meccaniche
*   Risolvi i Sudoku per avanzare nei livelli.
*   Hai un numero limitato di **Vite**. Errori definitivi riducono le vite.
*   Usa **Oggetti** dall'inventario (in alto a destra) per ottenere aiuti.
*   Accumula punti completando livelli per potenziare il tuo account nell'*Ancestor's Legacy*.

### Salvataggio
Per salvare e uscire durante una partita:
1.  Clicca l'icona dell'**Ingranaggio** (in alto a destra).
2.  Seleziona **"Save & Exit"**.
